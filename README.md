# Sports League Attendance Analysis

A Tableau dashboard comparing attendance across major U.S. sports leagues from 2023 to 2025.

## Project Overview

This project looks at attendance trends across five major sports leagues:

- NFL
- MLB
- NBA
- NHL
- MLS

The goal was to compare two different attendance metrics:

1. Average attendance per game
2. Total attendance per season

The result shows how the “winner” changes depending on how the data is measured.

## Key Insight

The NFL dominates average attendance per game because it has fewer games and larger stadium crowds.

MLB dominates total attendance because it plays far more games across the season.

Same data. Different metric. Different winner.
